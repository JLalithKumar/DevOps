Create Backup
Compress-Archive `
-Path .\customer-service `
-DestinationPath .\customer-backup.zip `
-Force

Upload to S3
aws s3 cp customer-backup.zip s3://lalith-ex26-backup-2026/

Verify Upload
aws s3 ls s3://lalith-ex26-backup-2026

Download Backup
aws s3 cp `
s3://lalith-ex26-backup-2026/customer-backup.zip `
.

Restore
Expand-Archive `
-Path customer-backup.zip `
-DestinationPath restored-project `
-Force

Verify Restored Files
dir restored-project\customer-service

Validate Restore
cd restored-project\customer-service

mvn clean package
